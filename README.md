# Contact Rolodex Application

## Project Description

The Contact Rolodex Application is a Java-based program that allows users to manage contact information in a structured and efficient way. The application simulates a digital Rolodex, where users can store, retrieve, update, and delete contact records.

## Features

* Add new contacts
* View all contacts
* Search for a contact
* Update existing contacts
* Delete contacts
* Persistent storage using a database (SQLite)

## Technologies Used

* Java
* SQLite
* VS Code

## Object-Oriented Concepts

* Inheritance (different contact types if implemented)
* Encapsulation (private fields with getters/setters)
* Constructors
* Abstraction (class structure and design)

## How to Run

1. Compile the Java files
2. Run the App.java file
3. Follow the menu prompts in the console
